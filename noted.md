## What is Git?
Git is a version control system that tracks changes in files so developers can collaborate and revert changes if needed.

## Git commands used today:
1. git init — initializes a new Git repository
2. git add — stages changes for commit
3. git commit — saves a snapshot of changes
4. git status — shows current state of repo
5. git push — sends changes to GitHub

## Confusion & resolution:
I was confused about staging vs committing. I learned that staging (git add) prepares changes, while committing (git commit) saves them permanently in history.
